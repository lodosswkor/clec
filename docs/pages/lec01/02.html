VS2019 다운로드 및 설치

비주얼 스튜디오 2019 다운로드 및 설치

VS2019 다운로드 URL : https://visualstudio.microsoft.com/ko/

1. Visual studio 2019 "Community" Edition 다운로드/실행

다운로드가 완료되면 "열기"를 클릭해 주세요

2. 설치항목 선택

Visual Studio는 기본으로 C/C++ 개발환경을 지원하지 않습니다.
따라서 설치항목 중 "C++을 사용한 Desktop 개발" 항목을 선택합니다.

하위항목 중 "v142 C/C++ 모듈(x64/x86)" 에 체크가 되어 있는지 확인합니다.
